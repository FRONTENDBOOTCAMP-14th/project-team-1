<h2>🔸설치/실행 방법</h2>

1. 프로젝트 폴더 클론

```
git clone https://github.com/FRONTENDBOOTCAMP-14th/project-team-1.git
```

2. 클론한 폴더로 이동하기
   
```
cd project-team-1
```

3. npm 패키지 설치

```
npm install
```

4. vite 개발 서버 실행
   
```
npm run dev
```
